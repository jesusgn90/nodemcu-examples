# Humidity and temperature reader using DHT11

Read values from a DHT11 sensor and print temperature (C, F) and humidity (%).

### Diagram

![diagram](./diagram.png)

### Requirements

```
1 x NodeMCU
1 x DHT11 (take care of resistor)
3 x wire (optional) / 1 x breadboard
1 x Micro USB (C)
1 x Laptop
```
